High School Management System - API Documentation

Base URL: https://your-domain.com/api/v1/

Authentication: JWT Bearer Token (Authorization: Bearer <token>)

Multi-Tenancy: Required header: x-tenant: <tenant-slug>

Authentication

POST /api/v1/auth/login/
User Login
Authenticate a user using email, username, or staff ID. Returns access and refresh tokens.
POST /api/v1/auth/token/refresh/
Refresh Access Token
Obtain a new access token using a valid refresh token.
GET /api/v1/auth/user/current/
Get Current User
Retrieve detailed information about the currently authenticated user.
POST /api/v1/auth/verify/
Verify Token
Validate a JWT token and check if it's still valid.
GET /api/v1/auth/
List Tenant Users
Get a list of all users within a specific tenant.

Core / Tenant Management

GET /api/v1/tenants/
List Tenants
Retrieve a list of all tenants/schools.
POST /api/v1/tenants/
Create Tenant
Create a new tenant/school with all necessary configuration.
GET PUT PATCH DELETE /api/v1/tenants/{id}/
Tenant Details
Get, update, partially update, or delete a specific tenant.
